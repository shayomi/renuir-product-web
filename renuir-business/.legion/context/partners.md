# Partners

## Integration Partner Types

### Hotels & Hospitality

When a guest forgets an item, hotel staff logs it via the Renuir API or dashboard.
The system automatically tries to match it against active lost item reports.

### Airports & Transport

High-volume lost item environments. API integration allows automatic ingestion
of found items into the Renuir matching system.

### Restaurants & Cafes

Lighter integration — simpler dashboard or QR-based submission flow.

### Other Institutions (Gyms, Offices, Events)

Self-serve API key + webhook-based integration.

## Integration Methods

- **REST API** — full programmatic access (for tech-enabled partners)
- **Business Dashboard** — manual UI for non-technical staff
- **QR Drop** — (planned) physical QR code at establishment for easy item submission
