# Skill: Component Creation
## When to use
Use this skill when implementing a new UI component for the Renuir Next.js web app.

## Steps (numbered)
1. Identify the target section and create the file in `components/<section>/`.
2. Build it as a server component by default unless client interactivity is required.
3. Name the component with PascalCase and keep file naming consistent.
4. Style using Tailwind utility classes only.
5. Export both a named export and a default export for the component.
6. Verify the component usage composes cleanly in the intended page/section.

## Output
A production-ready Next.js component in `components/<section>/` following server-first, PascalCase, Tailwind-only, and dual-export conventions.
