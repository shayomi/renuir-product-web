# Skill: Business Integration
## When to use
Use this skill when onboarding a new hotel, airport, restaurant, or institution to Renuir.

## Steps (numbered)
1. Create and provision the partner API key with scoped access.
2. Configure partner webhook endpoints and signing/verification settings.
3. Set up partner dashboard access for operational and fallback workflows.
4. Walk the partner through payload expectations and API error behavior.
5. Execute integration tests using sample and real-world scenarios.
6. Confirm successful event ingestion and handoff to ongoing support.

## Output
A fully onboarded partner with API key setup, webhook configuration, dashboard access, and validated end-to-end integration behavior.
