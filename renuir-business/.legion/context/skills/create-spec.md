# Skill: Create Spec
## When to use
Use this skill when starting any new feature, capability, or significant change.

## Steps (numbered)
1. Read `.legion/context/goals.md` and `.legion/context/priorities.md`.
2. Create a new folder using `.legion/specs/NN_feature-name/` numbering.
3. Copy `.legion/specs/templates/SPEC_TEMPLATE.md` into the new folder.
4. Copy `.legion/specs/templates/E2E_TEMPLATE.md` into the new folder.
5. Fill in all sections in both files with concrete, testable detail.
6. Update `.legion/context/projects.md` with the new feature status and ownership.

## Output
A complete numbered feature spec folder with both templates populated and project tracking updated.
